# kalman_filter

Kalman filter for tracking the position of an object of unknown dynamics. 

## Concept 

Kalman filter is a 

 `$z = x + y$`.

`$$a^2 + b^2 = c^2$$`

`$$\begin{vmatrix}a & b\\
c & d
\end{vmatrix}=ad-bc$$`
